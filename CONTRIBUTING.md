# Contributing to UAICP Website

This repository hosts the public documentation website for UAICP.

## Workflow

1. Open an issue for your proposed change.
2. Link the issue in your PR.
3. Keep content aligned with `uaicp_specification` source-of-truth docs.
4. Include validation notes (build/link/content checks).

## Definition of Done

- CI checks pass
- Content updates are technically accurate
- Navigation/sitemap remains coherent

## Local Private Notes

Use `.private/` for local-only working notes. Only `.private/.gitkeep` is tracked.
