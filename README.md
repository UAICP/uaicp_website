# UAICP Website

Public documentation site for UAICP.

UAICP is the open-source reliability contract for agentic workflows.

Canonical protocol scope and roadmap are maintained in:

- `https://github.com/UAICP/uaicp_specification`

## What this site contains

- specification docs
- integration guides for framework adapters
- use-case examples
- governance and contribution guides

## Local Development

```bash
npm install
npm run start
```

## Build

```bash
npm run build
npm run serve
```
